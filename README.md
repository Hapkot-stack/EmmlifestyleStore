## 🎯 Project Title
**Emmlifestyle Store – No-Code E-Commerce Website Design Using Wix**

---

## 🧾 Project Objective
The objective of this project is to design and develop a simple and functional e-commerce website using a no-code platform (Wix). The project demonstrates understanding of online store structure, product presentation, and basic customer shopping experience.

---

## 🛠️ Platform Used
- Wix (No-Code Website Builder)

## Website Features Implemented

-  Homepage with welcome message and store branding  
-  Product page with at least 5 fashion products  
-  About page with store description, mission, and vision  
-  Contact page with email, phone, and contact form  
-  Cart functionality (Wix add-to-cart system)  
-  Responsive design for mobile and desktop users
